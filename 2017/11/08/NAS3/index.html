Raspberry PI NAS konfigurálás - 3. Rész - Apt-Cron

2017-11-08
IT
debian/minibian/nas/raspberry pi/sysop

Idővel szükségessé vált számomra, hogy legyen egy a belső hálózatból, illetve publikusan is elérhető NAS-om. Ezt egy Raspberry PI2 (vagy 3)
ami a boltban kapható stock okos dobozok képességein felül egy komplett unix workstation álljon rendelkezésre.

cron-apt

A cron-apt egy olyan szolgáltatás, ami képes apt műveleteket cronon keresztül végrehajtani. Ezt automatikus frisíítések végett fogjuk alkalmazni. 

# apt-get install cron-apt

Eztán jöhet a konfigurálás. A konfigurációs file a /etc/cron-apt/config útvonal alatt talákható. Mindkét disztrón van vi.

# vi /etc/cron-apt/config

Idővel mélyebben belemegyek, de egyelőre legyen annyi, hogy (…) Adjuk az alábbi sort a filehoz:

MAILON="always"
